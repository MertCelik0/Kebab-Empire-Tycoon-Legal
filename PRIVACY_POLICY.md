## Information We Collect

### 1. Information You Provide to Us

We do **NOT** collect any personally identifiable information directly from you. The App does not require user registration or login.

### 2. Information Automatically Collected

When you use the App, we automatically collect certain information about your device and how you interact with the App:

#### a) Device Information
- Device type and model
- Operating system and version (iOS/Android)
- Unique device identifiers (IDFA on iOS, Advertising ID on Android)
- Device language and region settings
- Screen resolution and device orientation

#### b) Usage Information (via Firebase Analytics)
- Game progress and achievements
- Features used (equipment purchases, manager hires, prestige actions)
- In-app purchases and transaction history (non-financial details)
- Session duration and frequency
- Tutorial completion status
- World progression and boss event participation
- Quest and achievement completions
- VIP subscription status (tier and duration)

#### c) Performance Data (via Firebase Performance Monitoring)
- App launch time and performance metrics
- Screen rendering performance
- Network request latency
- Game operation performance (save/load times, etc.)
- Custom operation traces

#### d) Crash and Diagnostic Data (via Firebase Crashlytics)
- Crash reports and stack traces
- Error logs and exception data
- Device state at time of crash
- User actions leading to crashes
- App version and build number

#### e) Advertising Data (via Google AdMob)
- Ad impressions and interactions
- Ad preferences and interests (inferred)
- Device advertising ID
- IP address (temporary, for ad serving)
- General location (country/region level, derived from IP)

### 3. Local Storage

The App stores game progress and settings locally on your device using:
- AsyncStorage (encrypted local storage)
- This data never leaves your device unless you manually back it up via device backup systems (iCloud, Google Drive)
